# Search Engine (Skillbox Final Project)

## Описание проекта

Данный проект представляет собой консольный поисковый движок, разработанный в рамках финального задания курса.

Программа индексирует текстовые документы и выполняет поиск по ним на основе пользовательских запросов.

Результаты поиска ранжируются по релевантности и сохраняются в файл `answers.json`.

---

## Функциональность

- Индексация документов
- Поиск по словам
- Расчёт релевантности
- Ограничение количества результатов (`max_responses`)
- Работа с JSON файлами:
  - config.json
  - requests.json
  - answers.json
- Поддержка модульного тестирования (Google Test)

---

## Используемые технологии

- C++
- CMake
- nlohmann/json
- GoogleTest
